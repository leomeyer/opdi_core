WinOPDI - OPDI reference implementation for Windows 32 bit
see Open Protocol for Device Interaction

Project for Microsoft Visual Studio 2010

To build this project you will need the POCO library in the folder code\c\libraries\POCO.
Please see there for instructions.

To run the program you also have to copy the POCO DLLs to the binary folder (Debug for the Debug configuration).
Copy the following files from the code\c\libraries\POCO\bin folder:
PocoUtil.dll
PocoFoundation.dll
PocoFoundationd.dll
PocoNet.dll
PocoNetd.dll
