Hardware: Poladisc 1.2
Wizslave implementation for AVR Atmega168; supports four LED channels.
Has a default color scrolling algorithm which can be disabled.

Expects communication on the serial port (suitable for Bluetooth module).

WinAVR project (Programmer's Notepad 2). You can use this project as a template for other AVR based implementations.
