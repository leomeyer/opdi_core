AndroPDI Fernsteuerung

Danke, dass Sie AndroPDI Fernsteuerung benutzen!

Um anzufangen, tippen Sie bitte auf "Geräte verwalten".

Wenn Sie einen QR-Code für ein Zielgerät haben, können Sie das Gerät mit der Funktion "QR-Code scannen" hinzufügen.

Weitere Geräte werden hinzugefügt, indem Sie ihren Namen und Adresse angeben, sowohl für Bluetooth- als auch Netzwerkgeräte (TCP/IP).

Wenn Sie ein Gerät hinzugefügt haben, tippen Sie darauf, um sich zu verbinden. Für Bluetooth-Geräte müssen Sie gegebenenfalls ein Pairing durchführen; hierzu benötigen Sie eine PIN (meist 0000 oder 1234). Ist das Gerät verbunden, tippen Sie nochmals, um seine Steuerung anzuzeigen.

Für weitere Informationen wenden Sie sich bitte an leomeyer@gmx.de oder besuchen http://www.software-services.de/OPDI.

Viel Spaß!
