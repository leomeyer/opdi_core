AndroPDI Remote Device Controller

Thank you for using the AndroPDI Remote Controller!

To get started, tap on the "Manage devices" item.

If you have a QR code for the target device, it's simplest to use the "Scan QR code" function to add the device to the list.

Alternatively, you can add devices by entering their address and name, both for Bluetooth and networked (TCP/IP) devices.

Once you have added a device, tap it to connect to the device. For a Bluetooth device you will most likely need to perform pairing using a PIN (try 0000 or 1234 if in doubt). If connected, tap again to show its controls.

For more information, please contact leomeyer@gmx.de or visit http://www.software-services.de/OPDI.

Have fun!
